# OrangeHRM Automation Testing

## About This Project
This is my first Selenium automation project.
I have automated the login page of OrangeHRM demo website
using Selenium WebDriver with Java.

## What I Tested
- Valid login with correct username and password
- Invalid login with wrong username and password

## Technologies Used
- Java
- Selenium WebDriver 4.18.1
- Maven
- ChromeDriver

## How to Run
1. Clone this repository
2. Open in Eclipse as Maven Project
3. Run LoginTest.java as Java Application
